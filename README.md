# Node P2P chat

## Execution 

```bash
## Run program
PORT=<PORT> npm run start [...hosts]

## Run and watch program files changes
PORT=<PORT> npm run start:watch  [...hosts]


# Args:
# - PORT: Listening port
# - hosts: Peers to connect into
```
